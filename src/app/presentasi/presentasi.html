Apa aja fiturnya?

Catat & Atur Tugas

Kamu bisa menuliskan semua kerjaan yang harus dikerjakan, lalu mengatur mana yang paling penting.

Pengingat Otomatis

Aplikasi bakal ngasih tahu kamu kalau ada tugas yang udah mau mendekati deadline-nya.

Kelompokkan dengan Kategori

Tugas bisa dibagi-bagi misalnya "Kantor", "Kuliah", "Pribadi" biar nggak campur aduk.

Cari Tugas Langsung

Punya ratusan tugas? Tinggal ketik kata kuncinya, langsung ketemu tanpa scroll-scroll.

Tandai Sudah Selesai

Setiap tugas yang selesai bisa dicentang, biar kamu tahu sudah sejauh mana progresnya.

Data Tersimpan Aman

Semua data tugas tersimpan di perangkat kamu, nggak bakal hilang walau aplikasi ditutup.

Contoh kodenya

Gini cara kerja di balik layar — dikodein pakai JavaScript.

task.js
// Menambah tugas baru function addTask(title, priority) { const task = { id: Date.now(), title, priority, // "tinggi" | "sedang" | "rendah" done: false, createdAt: new Date() }; tasks.push(task); saveTasks(); // simpan ke localStorage renderTasks(); // tampilkan ke layar }
filter.js
// Filter & cari tugas function getFilteredTasks(keyword, status) { let result = [...tasks]; // Filter berdasarkan kata kunci if (keyword) { result = result.filter(t => t.title.toLowerCase() .includes(keyword) ); } // Filter: belum selesai if (status === "active") result = result.filter(t => !t.done); return result; }

Teknologi & Tools

Metode Pengembangan

Component-Based

Setiap bagian UI dibuat sebagai komponen terpisah yang bisa dipakai ulang.

Event-Driven

Aplikasi merespons aksi user (klik, ketik) secara real-time tanpa reload.

Separation of Concerns

Kode logika, tampilan, dan data dipisah biar mudah dikelola.

Mobile-First

Desain diutamakan untuk HP dulu, baru diadaptasi untuk layar lebih besar.
